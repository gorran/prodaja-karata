# Electron App

Create with [minimal Electron starter](https://github.com/jaredpalmer/electron-starter).

### Technologies

- Electron
- React
- Redux
- Webpack (Babel Stage-0, JSON-loader, File-loader for images)
- React Dev Tools
- Devtron

## Install

```
npm i
npm start
```
